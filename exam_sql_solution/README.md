# Решения заданий ГИА по базам данных

Папка содержит готовые материалы по модулям 1-8 из задания.

## Состав

- `sql/M1_install_and_users.sql` - установка структуры: пользователи `user1`-`user10`, базы `BD1`-`BD10`, база `BD` и таблица `Users`.
- `sql/M2_encrypt_passwords.sql` - шифрование паролей и безопасный просмотр расшифрованных значений.
- `sql/M3_backup_restore.sql` - резервное копирование и восстановление базы `BD`.
- `sql/M4_schema_and_seed.sql` - база для производства, 3НФ, связи, ключи и тестовые данные.
- `sql/M5_procedure_and_trigger.sql` - процедура отчета за период и триггер пересчета суммы заказа.
- `sql/M6_archive_orders.sql` - перенос заказов за выбранный месяц в архивную таблицу.
- `app/order_analyzer.py` - модуль анализа заказов с сортировкой, фильтрацией, поиском и итогами.
- `docs/user_guide.md` - руководство пользователя для модуля анализа.
- `docs/er_diagram.md` - текстовая ER-диаграмма и описание связей.

## Как запускать

1. Откройте SQL Server Management Studio.
2. Выполните скрипты из папки `sql` по порядку: `M1`, `M2`, `M3`, `M4`, `M5`, `M6`.
3. Для модуля 7 запустите:

```powershell
python app\order_analyzer.py
```

Приложение само создаст SQLite-базу `app\data\orders.db` с тестовыми заказами при первом запуске.
