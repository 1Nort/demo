# Руководство пользователя

## Назначение

Модуль `order_analyzer.py` позволяет анализировать заказы из созданной базы данных: сортировать записи, фильтровать их по клиенту, искать совпадения и видеть итоговое количество заказов и общую сумму.

## Запуск

1. Откройте папку `exam_sql_solution`.
2. Выполните команду:

```powershell
python app\order_analyzer.py
```

3. При первом запуске программа создаст тестовую базу `app\data\orders.db`.

## Работа с заказами

- Поле `Выберите заказчика` фильтрует таблицу по выбранному клиенту.
- Кнопка `Фильтровать` применяет выбранный фильтр.
- Кнопка `Показать все` отменяет фильтрацию.
- Поле `Введите строку поиска` используется для поиска по всем ячейкам таблицы.
- Кнопка `Найти` подсвечивает строки, где найдено совпадение.
- Блок сортировки меняет порядок записей по выбранному полю: по возрастанию или по убыванию.
- Внизу окна отображаются `Всего заказов` и `Общая сумма`.

## Обработка ошибок

Если база отсутствует, приложение создает ее автоматически. Если пользователь выбрал неверные параметры или поиск ничего не нашел, программа не завершается аварийно и продолжает работу.
